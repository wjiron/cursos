¿Que es HTML?

HyperText Markup Language es un lenguaje de marcado compuesto por elementos que representan un contenido especifico, HTML no se debe confundir con un lenguaje de programación.

Marcado significa que con HTML se declara que se presenta al espectador, mas no como es representado. Las representaciones visuales se hacen con CSS y son realizadas por los navegadores.

Elementos en HTML

Los documentos HTML están compuestos por elementos, donde un elemento usualmente consiste en una etiqueta de entrada, etiqueta de cierre, donde ambas contienen el nombre del elemento rodeado por corchete angulares y entre ellos existe un contenido

Existen elementos que no tienen etiquetas de cierre o contenido y se llaman elementos void, entre ellos se incluyen a:

  • <img>
  • <meta>
  • <link>
  • <input>

016 - Seccionamiento

Seccionamiento en HTML

Cuando necesitamos seccionar nuestros elementos HTML tendremos las etiquetas semanticas que visualmente no muestran nada, pero ordenaran nuestros elementos segun lo que necesitamos. Cabe mencionar que para el posicionamiento de nuestros elementos necesitamos de CSS.

Etiquetas Semánticas

  • <nav></nav>: representa una sección de una página que enlaza a otras páginas o a otras partes dentro de la página. Se conoce como barras de navegación.

  • <header></header>: representa el contenido introductorio para su contenido de seccionamiento, ubica la idea central de la página y de lo que va a contener.

  • <main></main>: este elemento contiene el contenido principal de su página web. Este contenido es exclusivo de la página individual, y no debe aparecer en ninguna otra parte del sitio.

  • <article></article>: este elemento contiene contenido autónomo como artículos, publicaciones de blog, comentarios de usuarios.

  • <section></section>: se utiliza para representar una sección general dentro de un documento.

  • <aside></aside>: representa una sección de la página que abarca un contenido relacionado con el contenido que lo rodea, por lo que se le puede considerar un contenido independiente.

  • <footer></footer>: Representa un pie de página para el elemento que lo contiene. Generalmente contiene información acerca de quién lo escribió, enlaces a documentos relacionados, datos de derechos de autor o similares.

Nota: estas etiquetas visualmente no muestran nada por si solas pero estructuran de forma correcta nuestro documento HTML. Agregando CSS se pueden ubicar y visualizar de acuerdo a lo que necesitemos.

018 - Labels

Label en HTML

El elemento label se utiliza para agregar un texto que hace referencia a un elemento de un formulario. Se agrega un label con el atributo for que tiene como valor el id del elemento input al cual se quiere hacer referencia. También se puede agregar un label sin referenciar, Sus etiquetas son:

<label> contenido del label </label>
